## 5.5 内置运算符

在Hive有四种类型的运算符：

* 关系运算符

* 算术运算符

* 逻辑运算符

* 复杂运算符

可以在Beeline或者Hive CLI交互窗口中，使用以下命令来查看最新的文档。

```
SHOW FUNCTIONS;
DESCRIBE FUNCTION <function_name>;
DESCRIBE FUNCTION EXTENDED <function_name>;
```
